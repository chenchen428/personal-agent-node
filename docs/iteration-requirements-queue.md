## 后续入队规则

1. 新需求追加到表格末尾，不插队重排。
2. 涉及 Agent 使用方式的任务修改，必须同时更新 Personal Agent Skill、CLI 和对应测试用例。
3. 涉及页面结构或交互的修改，必须同步 Prototype 设计稿和实现。
4. 每轮本地验收安装后，更新“本地交付”列及安装版本。
