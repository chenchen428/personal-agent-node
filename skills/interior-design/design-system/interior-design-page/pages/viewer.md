# Viewer override

This page is a quiet architectural workbench, not a property landing page. It overrides the generated master where they conflict.

- Palette: paper `#F7F8F6`, graphite `#202522`, drafting line `#D7DCD7`, moss `#315F4A`, oak `#C9A77B`, signal `#B94D32`.
- Typography: local system sans only; use tabular numerals for area and progress. No remote fonts.
- Composition: one edge-to-edge Three.js canvas, compact translucent top controls, bottom growth timeline. No hero copy, cards, pricing, gradients, or decorative shadows.
- Signature detail: the progress track reads like an architectural section line with room-stage ticks.
- Motion: object construction and camera movement communicate model state. No ornamental stagger or overshoot.
- Desktop: title/status left, view/light controls right, timeline bottom. Mobile: title/status top, two-row 44px toolbar and timeline above the safe area.
- Accessibility: 4.5:1 text contrast, visible 2px focus ring, aria labels, pointer and keyboard activation, `prefers-reduced-motion`, WebGL error message.
