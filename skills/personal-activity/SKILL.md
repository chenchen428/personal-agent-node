# Personal Activity

Use only the ephemeral Activity capability issued to the canonical main Agent for the current turn. Never display, persist, or delegate it.

Search Activity before session history when the user asks for an earlier result. Prefer one correlation-keyed evolving item over repeated progress posts. Create Activity only for meaningful user value, not tool calls, logs, heartbeats, token usage, or every status transition.

Keep titles within 30 visible characters and attachments within ten. Use stable governed targets and current-Space managed object IDs. Page Activity must target the exact `pageId` returned by Pages.

Read [activity.md](references/activity.md) for command forms, revision safety, attachment rules, and main-Agent isolation.
