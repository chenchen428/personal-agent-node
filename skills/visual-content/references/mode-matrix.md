# Visual Mode Matrix

## Core Dimensions

| Dimension | Choices | Decision rule |
|---|---|---|
| Information type | scene, comparison, flow, framework, timeline, hierarchy, data | Match the relationship the viewer must understand |
| Density | sparse, balanced, dense | Match viewing distance and platform, not amount of source text |
| Rendering | flat, editorial, hand-drawn, painterly, technical, photographic, pixel | Match subject and audience expectations |
| Palette | neutral plus accent, warm, cool, vivid, monochrome, retro | Preserve contrast and semantic color meaning |
| Text | none, title, title plus labels, text-rich | Minimize bitmap text when exact copy matters |

## Mode Guidance

### Cover

Use one focal idea, a clear title zone, and intentional negative space. Default to 16:9 for general articles, 2.35:1 for cinematic banners, 1:1 for square feeds, and platform-specific dimensions when provided.

### Article Illustration

Illustrate only sections where a visual adds explanation, comparison, atmosphere, or memory. Prefer a small coherent series over one image per heading.

### Infographic

Select the layout from the relationship: comparison table, funnel, timeline, hierarchy/tree, quadrants, layers, circular flow, journey, or annotated feature map. Keep one primary reading path.

### Diagram

Use flowchart for steps, sequence for time-ordered interactions, structural for containment/topology, class for types and relations, and illustrative for mechanism intuition. Use SVG when labels and connectors must be exact.

### Slide Deck

Build a narrative arc: opening promise, context/problem, evidence, mechanism/options, recommendation, close. One message per slide; vary layouts while keeping the system stable.

### Comic

Define characters, learning goal, page/panel beats, dialogue limits, and continuity. Educational clarity outranks visual spectacle.

### Social Series

Use a cover, progressive content cards, and a close. Optimize for phone scanning. Assign one clear point per card unless the requested mode is intentionally dense.
